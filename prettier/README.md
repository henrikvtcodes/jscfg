# prettier-config-henrik

### How to use

**Installation**

```sh
# pnpm
pnpm add -D prettier prettier-config-henrik
# Bun
bun add -d prettier prettier-config-henrik
```

**Usage**

```js
import sharedConfig from "prettier-config-henrik";

const config = {
  ...sharedConfig,
  // your config here
};

export default config;
```
